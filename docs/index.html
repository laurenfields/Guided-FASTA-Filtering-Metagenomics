Guided FASTA Filtering — Metaproteomics

Build a tailored protein FASTA for a metagenome sample from biological context — KEGG Orthology (KO) terms. You supply the sample's KO annotation table and its predicted-protein FASTA; the KO → gene join and FASTA filtering run in your browser and nothing is uploaded. Every run produces a reproducibility manifest.
